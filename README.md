# E-Commerce Store (Angular)

A modern, scalable e-commerce web application built with Angular.  
It provides a complete shopping experience including product browsing, cart management, wishlist, authentication, and secure payments.

The project is designed with performance, SEO, accessibility, and maintainability in mind, following modern frontend architecture best practices.

---

## Features

- Product catalog with detailed product pages
- Shopping cart functionality
- Wishlist (favorites) system
- User authentication and authorization
- Secure checkout integration (Stripe)
- Admin dashboard for product management
- Server-Side Rendering (SSR) for SEO optimization
- Responsive and mobile-first UI design

---

## Tech Stack

- Angular (SSR enabled with Angular Universal)
- Supabase (Authentication & Database)
- Stripe (Payment processing)
- NgRx (Global State management)
- Angular Signals (UI state handling)

---

## Goals

- Provide a production-ready e-commerce architecture
- Ensure scalability and maintainability
- Follow best practices for SEO and accessibility
- Demonstrate modern Angular architecture patterns
